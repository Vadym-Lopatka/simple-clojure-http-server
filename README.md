# Showcase example of how to build HTTP endpoints

## Tech stack: 
Clojure, Ring, Reitit and Muuntaja

## Functionality:
- '/' Return string on base URL
- '/users' POST create a user
- '/users' GET Return list of users
- '/users/:id' GET Return user by id
